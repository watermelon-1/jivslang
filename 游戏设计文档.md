# 鸡与狼 - 游戏设计文档

## 一、游戏概述

一款自动对战模拟游戏。鸡和狼在一个封闭的鸡舍中自动移动和攻击，直到一方血量归零，游戏结束。

---

## 二、游戏场景

| 属性 | 数值 |
|------|------|
| 场景形状 | 正方形 |
| 场景尺寸 | 10米 × 10米 |
| 边界行为 | 实体碰到墙壁后按光的反射定律反弹 |

## 五、运动系统

### 5.1 初始状态

- 鸡和狼的初始位置在 10m × 10m 范围内 **随机生成**。
- 需要考虑两个实体初始时不重叠（最小间距应大于各自占地面积之和的一半）。
- 初始运动方向（速度向量）也是 **随机** 的。

### 5.2 运动规则

1. 每个实体沿直线匀速移动。
2. 碰到墙壁时，按 **光的反射定律** 反弹：
   - 碰到左/右墙 → 水平速度分量反向
   - 碰到上/下墙 → 垂直速度分量反向
   - 碰到角落 → 两个分量同时反向
3. 实体视为有体积的矩形（2m × 2m），以实体中心为坐标参考点，碰撞检测需要考虑实体的半宽/半高与墙壁的距离。

### 5.3 实体间碰撞

- 当鸡和狼发生碰撞时，触发狼的攻击（鸡掉血 30）。
- 碰撞后双方 **仅分离位置**（防止重叠），**速度保持不变**。

## 六、游戏流程

```
游戏开始
  ↓
初始化场景（10m × 10m）
  ↓
随机生成鸡的初始位置和速度方向
  ↓
随机生成狼的初始位置和速度方向（确保与鸡不重叠）
  ↓
游戏主循环（每帧更新）：
  ├─ 更新鸡和狼的位置（沿速度方向移动，速度 1m/s）
  ├─ 检测墙壁碰撞 → 按光反射定律反弹
  ├─ 检测鸡与狼碰撞 → 狼攻击（鸡-30HP）+ 弹开
  ├─ 鸡攻击计时器 → 每1秒朝狼方向发射一枚鸡蛋
  ├─ 更新所有鸡蛋位置（飞行中）
  ├─ 鸡蛋碰墙检测 → 鸡蛋消失
  ├─ 鸡蛋命中狼检测 → 狼-10HP，鸡蛋消失
  ├─ 狼激光计时器 → 每5秒狼静止，咆哮发射激光
  ├─ 激光命中鸡检测 → 鸡-50HP
  ├─ 检查胜负条件
  └─ 渲染画面
  ↓
游戏结束
  ├─ 所有鸡死亡 且 无存活孵化蛋 且 狼血量 > 0 → 狼获胜
  ├─ 狼血量 ≤ 0 → 鸡获胜
  └─ 所有鸡死亡 且 无存活孵化蛋 且 狼血量 ≤ 0 → 平局
```

## 七、后续可扩展点

| 扩展方向 | 说明 |
|----------|------|
| 可配置参数 | 血量、攻击力、攻击频率、移动速度、实体大小等全部可调 |
| 实体间碰撞物理 | 鸡和狼碰撞后的弹开效果 |
| 多种实体 | 增加更多鸡/狼，或新增角色类型 |
| 特殊技能 | 如鸡的闪避、狼的加速冲刺等 |
| 地图障碍物 | 在鸡舍中添加障碍物，影响移动和反弹 |
| 鸡蛋飞行轨迹 | 考虑鸡蛋是否可以碰墙反弹，或者碰墙即消失 |
| UI/HUD | 实时显示血量条、计时器、伤害数字飘字等 |
| 统计面板 | 回合数、命中率、胜负记录等 |
| 音效 | 攻击音效、碰撞音效、胜利/失败音效 |
| 难度设置 | 调整狼的移动速度、攻击力等 |
